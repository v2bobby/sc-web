# SabajCore — Official Website

**Next-Generation PropTech Infrastructure & Urban Development for Nigeria**

A high-performance, modern single-page website built to generate excitement and collect early adopters for **SabajCore** before the MVP launches.

Live Demo: [sabajcore.github.io](https://sabajcore.github.io) *(after deployment)*

![SabajCore Hero](https://via.placeholder.com/1200x600/0a1729/10b981?text=SABAJCORE+HERO)

---

## ✨ Features

- **Fully Responsive** — Mobile-first design with perfect Tailwind aesthetics
- **High-end Branding** — Custom SVG logo with emerald green / dark theme
- **Smooth Animations** — Scroll-triggered interactions, hover effects, and modal
- **Waitlist Modal** — Functional form with success toast
- **Fast & Lightweight** — No heavy frameworks, pure HTML + Tailwind + vanilla JS
- **SEO Optimized** — Proper meta tags and semantic structure
- **Production Ready** — Ready for GitHub Pages, Vercel, or Netlify

---

## 🎨 Design & Branding

- **Primary Color**: Emerald-400 (`#10b981`)
- **Typography**: `Space Grotesk` (headings) + `Inter` (body)
- **Theme**: Dark cyber-proptech with premium emerald accents
- **Logo**: Custom SVG included (`logo.svg`)
